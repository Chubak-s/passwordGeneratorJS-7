# Генератор паролей
## Описание
Был создан веб-сайт генератор случайных паролей. Это приложение генерирует надежный пароль с прописными и строчными буквами, цифрами и символами.
Также добавлен значок копирования пароля, чтобы скопировать новый сгенерированный пароль. Доступно по ссылке: https://chubak-s.github.io/passwordGeneratorJS-7/
## Технологии
*HTML5
*CSS
*DOM
*JavaScript
